4.条件里面套条件

思考：使用python程序实现3数排大小
153，142, 19 高 中 低
思路：
1.先找出最大数
2.和-最小数-最大数= 中间数
实现代码：

实现 三数 排序 高 中 低

num1=123;
num2=234;
num3=14;

max1=0;#假设最大值 初始值0
min1=0;#最小值 0

#统计三数和
sum1=num1+num2+num3;

if num1>num2 and num1>num3:
max1=num1;

if num2>num3:
    min1=num3;
else:
    min1=num2;

elif num2>num1 and num2>num3:
max1=num2;
if num1>num3:
min1=num3;
else:
min1=num1;
else:
max1=num3;
if num1>num2:
min1=num2;
else:
min1=num1;
#最大值
print(max1)
#中间值
print(sum1-max1-min1)
#最小值
print(min1)

循环结构
什么是循环？
围着操场跑100圈

初始化语句 0
重复的事情 跑圈
循环的条件 0<100
控制循环 每跑一圈，圈数+1 （没有这个,就会变成死循环！）

循环1-10：
#初始化语句
a=1;
#循环的条件
while a<=10:
#循环体语句
print(a);
#控制循环语句
a=a+1;

语法：
初始化语句；
While 循环的判断条件:
循环体语句；
控制循环语句；
执行顺序：
初始化语句；
循环的判断条件
True:
循环体语句；
控制循环语句
False:
循环结束了！

案例：循环1-10奇数

案例：
石头剪刀布的案例（1-石头 2-剪刀 3-布）
规则： 石头>剪刀 剪刀>布 布>石头
人： 1- 2-3
机器： 1–3

条件结构
人>机器： 人胜利 1 and 2 or 2 3 or 3 1
机器>人 机器胜利
人=机器 平局

问题：
如何判断人、机器出拳？——再来判断！！！

知识点：
键盘录入——人

随机数——机器
