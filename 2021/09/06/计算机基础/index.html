    一、计算机硬件的介绍

1.什么是计算机？
计算机（computer）俗称电脑，是现代一种用于高速计算的电子计算机器，可以进行数值计算、逻辑计算，还具有存储记忆功能。是能够按照程序运行，自动、高速处理海量数据的现代化智能电子设备。
我们的生活、工作中都离不开计算机。

2.计算机发展历史
1)第一台计算机（晶体管）
1946年-1953年，美国宾夕法尼亚大学研制的人类历史上真正意义的第一台电子计算机，占地170平方米，耗电150千瓦，造价48万美元，每秒可执行5000次加法或400次乘法运算。共使用了18000个电子管

2)第二代计算机（晶体管）
1954年-1963年，IBM公司制造的第一台使用晶体管的计算机，体积也小的很多，同时还增加了浮点运算，使计算能力有了很大提高。

3)第三代计算机（集成电路）
1964年-1971年，多个元件集成到单一的半导体芯片上，让计算机变得更小，功耗更低，速度更快，在这一时期的发展还出现了操作系统，让计算机在中心程序的控制协调下可以同时运行多个不同程序。

4)第四代计算机（大规模集成电路）
1972年-至今，这是IBM的更新换代的重要产品，采用了大规模集成电路代替磁芯存储，小规模集成电路作为逻辑元件，并使用虚拟存储器技术，将硬件和软件分离开来，从而明确了软件的价值。

1983年第一台使用了鼠标的电脑，第一台使用图形用户界面的电脑。

计算机可分为超级计算机、工业控制计算机、网络计算机、个人计算机、嵌入式计算机五类。其中个人计算机是我们接触最多的，个人计算机主要分为以下三类：

4.计算机硬件的组成
硬件主要指计算机完成输入、存储、处理、输出功能的具体设备。计算机的硬件主要包括输入设备、输出设备、中央处理器、存储器、其他设备等组成。
1)输入设备：
输入设备——主要通过设备向计算机发送指令。
主要有：键盘、鼠标、扫描仪、摄像头等；

2)输出设备
输出设备——主要通过设备来查看信息和处理数据。计算机的输出设备主要有：显示器、打印机、音箱等；

3)CPU（中央处理器）
CPU是中央处理器，是计算机的核心部件，相当于人的大脑，它执行计算机所有的处理工作，因为cpu通常是决定了一台计算机的性能核心部件，通常人们以它来判断一台计算机设备的档次。

4)存储器
存储器（Memory）是现代信息技术中用于保存信息的记忆设备。主要包括硬盘、光盘、u盘、存储卡等。

Ps：在这里区分一下内存条与硬盘的区别：
存储方式不同：电脑硬盘上的数据是写入在硬盘内的，当电脑关机之后，写入的数据还存在电脑硬盘之内，重新开机即可读取之前写入的数据。内存条的数据是暂时性的存储在内存里的，当电脑关机之后数据即被清空，重新开机之后，内存里的数据都没有了。
存储容量不同：常用的电脑硬盘的存储容量从几十GB到几千GB（一般称作TB）不等。常用的内存容量则在几GB到几十GB不等。
5)其他设备
其他的设备：列如主板、机箱等。
主板：主板，又叫主机板、系统板、或母板，是计算机最基本的同时也是最重要的部件之一。主板一般为矩形电路板，上面安装了组成计算机的主要电路系统。

二、计算机软件的介绍
1.计算机软件的定义
计算机软件( Software，也称软件)是指计算机系统中的程序及其文档，程序是计算任务的处理对象和处理规则的描述；文档是为了便于了解程序所需的阐明性资料。程序必须装入机器内部才能工作，文档一般是给人看的，不一定装入机器。
2.计算机软件的分类
系统软件：控制和支持计算机系统，如windows、dos、Linux、mac os等;
应用软件：微信、QQ、LOL等；

3.常用工具软件的分类
压缩软件：可以将多个文件打包成一个文件，减少数据的存储空间；
列如：WinZip、好压
截图软件：FastStone Capture、snipaste等
办公软件：office、wps等
网页浏览器：IE、FireFox、chrome等
杀毒软件：火绒、360、电脑管家等

三、常见的dos命令
1.什么是DOS操作系统？
磁盘操作系统（Disk Operating System），是早期个人计算机上的一类操作系统。
从1981年MS-DOS1.0直到1995年MS-DOS 6.22的15年间，DOS作为微软公司在个人计算机上使用的一个操作系统载体，推出了多个版本。DOS在IBM PC 兼容机市场中占有举足轻重的地位。可以直接操纵管理硬盘的文件，以DOS的形式运行。
微软图形界面操作系统Windows NT问世以来，DOS是一个后台程序的形式出现的。名为Windows命令提示符。可以通过点击运行-CMD进入。

2.为什么学习DOS命令?
1、DOS能够Windows下无法完成的任务，例如对硬盘的初始化，CHOST下系统的恢复和备份。2、能够更加高效的对Windows进行管理，例如通过Ping命令来了解网络的连通状况。3、当Windows出现故障时，通过DOS能更加快速的对其进行修复等。
所以说，DOS具有高效，能快速处理常见故障的特点，深入学习DOS，有助于我们对电脑有更加深入的了解，Windows下做不了事，DOS来完成，特别是一些从事网络管理工作的人来说，学会了DOS，对于服务器的配置和网络故障的排除有很大的帮助。既能提高工作效率，又为我们有了进一步的技能提升。

四、计算机网络介绍
1．网络的概念
网络：一组相互连接的计算机多台计算机组成，使用物理线路进行连接，网络的优点、交换数据、资源共享。

2．网络的分类
计算机网络覆盖的地理区域决定了它的类型。

局域网：局限在小的地理区域内或单独的建筑物内范围（几十米～几公里），传输速度快、连接设备为双绞线、集线器、交换机等

城域网：规模局限在一个城市的范围内区域(几公里～几十公里)，连接设备为光纤、路由器。

广域网：网络覆盖范围跨越国界、洲界，甚至全球范围 ，典型代表：Internet（英特网），连接设备为海底光缆。
3．Internet(因特网)的简介
Internet(因特网) ：一个巨大的网络集合，Internet提供的常见的服务为：远程登录、文件传输、电子邮件、WWW（万维网）等。

4．WWW(万维网)的介绍
万维网WWW是World Wide Web的简称，也称为Web、3W等。WWW是基于客户机/服务器方式的信息发现技术和超文本技术的综合。
客户端：向另一台计算机请求某种服务的计算机
服务器：为其他计算机的请求提供服务的计算机
5．浏览器/服务器模型
浏览器/服务器(Brower/Server,B/S)结构，浏览器将请求发送给Web服务器，Web服务器对请求进行处理，将响应发回浏览器。
例如：打开浏览器，输入百度的网址访问，浏览器将请求发给百度的服务器，然后百度服务器收到请求后处理，最后在浏览器的页面上面响应出对应的百度网页出来。

6．客户端/服务器模型
客户端-服务器，即Client-Server(C/S)结构，客户端向服务器发出请求，服务器处理请求并将响应发送给客户端。
客户端（Client）或称为用户端，是指与服务器相对应，为客户提供本地服务的程序。
列如：你跟朋友微信聊天，双方首先得需要安装一个微信客户端，然后才能通过网络将双方的消息发送到微信服务器，最后再通过微信服务器把对方消息响应过来。
7．Ip地址的认识
1)什么是ip地址？
Ip地址用来标识网络上面的每台计算机，相当于计算机在网络里面的身份id。
列如：给人寄信的时候，邮件抬头写明收件人地址，方便邮递员投递。邮件末尾写明发件人地址，对方好回信。所以两台计算机要建立起通信，必须要有双方的地址。

Ps：查看自己的本机ip地址，进入dos窗口输入：ipconfig 回车即可

2)Ip地址如何进行分类？
IP地址 = 网络ID +主机ID
网络ID：标识计算机或网络设备所在的网段
主机ID：标识特定主机或网络设备

地址类用于指定网络 ID 并在网络 ID 和主机 ID 之间提供分隔方法，IANA（The Internet Assigned Numbers Authority，互联网数字分配机构)负责分配A、B、C类网络地址，具体主机地址由机构组织自行分配。

3)那些特殊的ip地址？
0.0.0.0：本机
127.0.0.1：本机回环地址，用于本机测试
255.255.255.255：当前子网，一般用于向当前子网广播信息
4)子网掩码
子网掩码：用于获取主机IP地址的网络ID，用于区分主机通信的不同情况，选择不同路径
A类默认子网掩码：255.0.0.0
B类默认子网掩码：255.255.0.0
C类默认子网掩码：255.255.255.0

五、进制换算
1.认识数制系统
也许你曾听过计算机是由0和1组成的这句话，实际上计算机使用的是二进制系统数制系统，其所能识别的指令代码和机器码都是由0和1组成的序列，这跟计算机内部的电子元件的结构有关，电子元件的结构状态(通电和断电)，使用二进制更容易解释其电子元件的状态。
但是我们生活中使用的是十进制数制系统(逢十进一)，如何把我们生活中的数字让计算机去理解，这就需要学习数制系统转换。
数制是用一种固定的符号和统一的规则来表示数值的方法，就是一种计数的规则。常见的数值系统有二进制、八进制、十进制、十六进制等。
2.二进制与十进制转换
二进制的组成范围：只能是0和1组成——》列如：101、1101
十进制的组成范围：只能是0到9组成——》列如：10、55

2)二进制转十进制:

计算规则：从右边往左边数，列如第1个数1是第5位，则其为1*2的5-1次方，依次类推，然后计算出其和，则该和就是对应的十进制。
3.八进制与十六进制
八进制的组成范围：为0——7组成——》列如：17可以为一个八进制数，但是85不是一个八进制数。
十六进制的组成范围：为0-9、ABCDEF组成,共16位。列如2A5是一个十六进制的数。
关于八进制、十六进制跟十进制的转换，可以参照二进制，把2换成对应8和16即可。
